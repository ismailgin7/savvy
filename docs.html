Dokumentasi Savvy

Panduan lengkap setup dan penggunaan aplikasi keuangan keluarga Savvy dari nol hingga siap pakai.

✅ Estimasi waktu setup: 15–30 menit. Tidak perlu keahlian coding khusus.

Pengantar

Savvy adalah aplikasi keuangan keluarga berbasis web yang dibangun menggunakan HTML, CSS, dan JavaScript murni — tanpa framework yang rumit. Data tersimpan di Firebase Realtime Database milikmu sendiri, sehingga privasi dan keamanan data sepenuhnya ada di tanganmu.

Yang kamu dapatkan

  • Source code lengkap (3 file: index.html, style.css, app.js)
  • Dokumentasi setup ini
  • Support via WhatsApp selama 30 hari

1. Setup Firebase

Firebase adalah layanan dari Google yang menyediakan database dan autentikasi untuk aplikasi Savvy.

Buat Project Firebase

1
Buka Firebase Console
Kunjungi console.firebase.google.com dan login dengan akun Google.
2
Buat project baru
Klik Create a project → isi nama project (contoh: savvy-keluarga) → disable Google Analytics → klik Create project.
3
Buat Realtime Database
Di menu kiri klik Build → Realtime Database → Create Database. Pilih lokasi Singapore. Pilih Start in test mode → Enable.
4
Aktifkan Authentication
Klik Build → Authentication → Get started → Email/Password → Enable → Save.
5
Tambah user
Di tab Users → Add user. Masukkan email dan password untuk setiap anggota keluarga yang akan menggunakan aplikasi.
6
Ambil konfigurasi
Klik ikon ⚙️ → Project settings → Your apps → klik ikon Web (</>). Isi nama app → Register app. Salin kode firebaseConfig yang muncul.
7
Update Firebase Rules
Di Realtime Database → tab Rules, ganti isinya dengan:
{
  "rules": {
    ".read": "auth != null",
    ".write": "auth != null"
  }
}

Klik Publish.

2. Konfigurasi Kode

Setelah Firebase siap, masukkan konfigurasi ke dalam file app.js.

1
Buka file app.js
Buka file app.js menggunakan VS Code atau editor teks lainnya.
2
Ganti firebaseConfig
Cari bagian const firebaseConfig di baris paling atas, ganti seluruh isinya dengan konfigurasi dari Firebase Console kamu:
const firebaseConfig = {
  apiKey: "ISI_DENGAN_API_KEY_KAMU",
  authDomain: "NAMA_PROJECT.firebaseapp.com",
  databaseURL: "https://NAMA_PROJECT-default-rtdb.REGION.firebasedatabase.app",
  projectId: "NAMA_PROJECT",
  storageBucket: "NAMA_PROJECT.firebasestorage.app",
  messagingSenderId: "ANGKA_ID_KAMU",
  appId: "APP_ID_KAMU"
};
⚠️ Jangan bagikan firebaseConfig ke orang lain. Simpan dengan aman.

3. Deploy ke GitHub Pages

1
Buat repository GitHub
Buka github.com → klik + → New repository. Beri nama (contoh: savvy-app) → pilih Public → Create.
2
Upload file
Di halaman repository, klik uploading an existing file. Drag & drop ketiga file (index.html, style.css, app.js) → klik Commit changes.
3
Aktifkan GitHub Pages
Klik tab Settings → Pages → Branch: main → Save. Tunggu 2 menit.
4
Buka aplikasi
Aplikasi bisa diakses di: https://usernamegithub.github.io/nama-repository
🎉 Selesai! Aplikasi Savvy sudah online dan bisa diakses dari HP maupun laptop.

Login & Akun

Gunakan email dan password yang sudah didaftarkan di Firebase Authentication untuk login. Kamu bisa tambah anggota keluarga dengan menambahkan user baru di Firebase Console → Authentication → Users → Add user.

Transaksi

Buka tab Transaksi untuk mencatat pemasukan, pengeluaran, atau transfer antar rekening.

  • Pemasukan — gaji, usaha, investasi, dll
  • Pengeluaran — belanja, tagihan, makan, dll
  • Transfer — pindah saldo antar rekening/dompet

Gunakan filter Semua / Per Bulan / Rentang / Tanggal untuk melihat transaksi berdasarkan periode tertentu.

Anggaran

Buka tab Anggaran untuk set batas pengeluaran per kategori tiap bulan. Anggaran akan otomatis reset di awal bulan baru. Warna progress bar:

  • 🟢 Hijau — masih aman (di bawah 80%)
  • 🟡 Kuning — hampir habis (80–99%)
  • 🔴 Merah — melebihi batas

FAQ

Apakah data saya aman?

Ya. Data tersimpan di Firebase milikmu sendiri — bukan di server pihak ketiga. Hanya user yang login yang bisa mengakses data.

Apakah bisa dipakai offline?

Tidak sepenuhnya. Aplikasi butuh koneksi internet untuk sinkronisasi data dengan Firebase. Namun data yang sudah dimuat akan tetap terlihat sementara meski koneksi terputus.

Berapa banyak user yang bisa login?

Tidak terbatas. Firebase gratis mendukung hingga ribuan user. Untuk keluarga, cukup tambahkan email setiap anggota di Firebase Authentication.

Apakah bisa ganti nama aplikasi?

Bisa. Cari dan ganti semua teks "Savvy" dan simbol "◈" di file index.html dan app.js dengan nama yang kamu inginkan.

Bagaimana kalau ada error?

Buka browser → klik kanan → Inspect → Console. Screenshot pesan error dan kirim ke support.
